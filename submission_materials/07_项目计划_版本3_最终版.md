# 云南省企业就业失业数据采集系统项目计划（最终版）

## 1. 项目概述
- 项目名称：云南省企业就业失业数据采集系统
- 项目目标：建设一个面向省、市、企业三级用户的就业失业数据采集与管理平台，实现备案、填报、审核、统计、维护、导出和提交材料交付。
- 开发方式：采用 Agent 辅助编程方式推进需求分析、开发、联调和材料整理。
- 配置管理方式：Git + GitHub。

## 2. 建设范围
- 用户与权限：省级、市级、企业三级用户登录与 RBAC 控制。
- 企业备案：备案录入、备案审核、备案查询、备案导出。
- 月报填报：就业人数填报、减少类型与原因校验、历史查询。
- 审核流程：企业提交、市级审核、省级终审、退回修改、上报部级。
- 数据管理：修订、删除、汇总统计、对比分析、趋势分析、导出交换。
- 系统维护：上报时限、用户管理、角色管理、系统监控。
- 前端展示：企业工作台、市级工作台、省级统计总览和省级综合管理。

## 3. 技术路线
- 后端：FastAPI + SQLAlchemy 2.0 + Pydantic v2 + JWT
- 数据库：PostgreSQL / MySQL 兼容，本地 SQLite 联调
- 前端：Vue 3 + Vite + Element Plus + Axios + ECharts
- 部署与迁移：Alembic + Conda + 一键启动脚本

## 4. 主要里程碑
- 里程碑 1：完成需求梳理与模型设计
- 里程碑 2：完成后端核心业务接口
