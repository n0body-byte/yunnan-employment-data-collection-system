# 测试与验收说明

## 1. 测试范围
本项目测试覆盖以下内容：
- 后端数据模型与 Schema 校验
- JWT 登录与密码修改相关安全逻辑
- 企业备案、月报填报、三级审核等主流程
- RBAC 权限范围控制
- 上报时限、系统监控、数据导出
- 前端页面构建与工作台重构后的可用性验证

## 2. 已执行测试
### 后端测试
执行命令：
```powershell
python -m unittest tests.test_schemas tests.test_security
```

验证内容包括：
- 组织机构代码、电话、邮编、月份格式等字段校验
- 月报跨字段逻辑校验
- 登录与密码相关安全逻辑
- ORM 与 Schema 的基础序列化能力

### 前端测试
执行命令：
```powershell
cd frontend
npm run build
```

验证内容包括：
- Vue 页面编译通过
- 路由懒加载配置有效
- 统一工作台页面构建通过
- Element Plus 按需引入配置有效
- ECharts 按模块导入后构建通过

## 3. 人工验收项
### 企业端
- 登录后进入企业工作台
- 提交备案信息
- 提交月报
- 人数下降时自动显示并校验减少原因
- 查看历史记录与通知

### 市级端
- 查看待审月报
- 审核通过或退回
- 发布和删除通知
- 修改密码

### 省级端
- 审核企业备案
- 月报终审、退回修改、数据修订
- 查看统计总览和图表
- 用户管理、角色管理、通知管理
- 配置上报时限
- 查看系统监控
- 导出 Excel
- 查看数据交换日志

## 4. 前端重构后的新增验收项
- 登录页样式与工作台风格统一
- 企业、市级、省级工作台均可在窄屏下正常显示
- 侧边栏在小屏下可收起并通过菜单按钮展开
- 省级综合管理页模块切换正常
- 图表组件在窗口缩放时可以自动重绘

## 5. 当前验收结论
项目当前版本已达到课程演示与提交要求，主流程功能可运行，前后端均可构建和启动。最近一轮前端优化后，系统已具备：
- 统一后台风格
- 响应式布局
- 路由懒加载
- 基础拆包优化
- Element Plus 与 ECharts 的按需优化

## 6. 说明
- 当前构建仍有大包提示，主要来源于 `element-plus` 和部分图表依赖，但已不影响项目运行与展示
- 部分旧页面已被统一工作台替代，现保留为引导页或兼容入口
