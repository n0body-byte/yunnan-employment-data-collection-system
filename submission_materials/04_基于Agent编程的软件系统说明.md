# 基于 Agent 编程的软件系统说明

## 1. 系统名称
云南省企业就业失业数据采集系统

## 2. 系统功能概述
本系统面向省级、市级和企业三级用户，提供企业备案、月度就业数据填报、分级审核、统计分析、Excel 导出、上报时限配置、系统监控和角色权限控制等功能。

## 3. 技术架构
- 后端：FastAPI + SQLAlchemy 2.0 + Pydantic v2 + JWT
- 前端：Vue 3 + Vite + Element Plus + Axios + ECharts
- 数据库：PostgreSQL / MySQL 兼容设计，本地 SQLite 联调
- 运维与交付：Alembic、Conda、一键启动批处理脚本

## 4. 核心模块
- 用户认证与 RBAC 权限控制
- 企业备案管理
- 月度数据填报与动态校验
- 市级 / 省级审核流转
- 省级统计汇总与图表可视化
- Excel 导出
- 系统维护与监控

## 5. Agent 参与方式说明
本项目采用 Agent 辅助编程模式推进开发，主要体现在以下方面：
- 根据文字需求快速拆分模块并生成实现计划
- 生成后端数据模型、Schema、路由与业务校验代码
- 生成前端 Vue 页面、表单规则和 ECharts 组件
- 辅助补充测试、部署脚本、依赖文件与 README 文档
- 在需求变更后快速调整实现并补充交付材料

## 6. Agent 编程带来的效果
- 缩短了从需求到代码原型的实现时间
- 提高了跨后端、前端、脚本和文档的联动效率
- 对频繁变更的课程项目任务具有更高适应性

## 7. 系统演示建议流程
1. 使用企业账号登录，提交企业备案信息
2. 使用企业账号填报月度数据，演示减少原因动态显示
3. 使用市级账号审核月报
4. 使用省级账号终审、查看统计图表并导出 Excel
5. 进入系统维护页面，演示上报时限设置和服务器监控

## 9. 项目源码组成
- app：FastAPI 后端代码
- frontend：Vue 前端代码
- alembic：数据库迁移脚本
- tests：测试代码
- scripts：演示数据初始化脚本
- submission_materials：课程提交材料
